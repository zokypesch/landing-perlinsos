About This Style Guide

This comprehensive style guide showcases all components available in the Perlinsos CSS framework. The design system is built around the Indonesian Independence Day theme using red, gold, and white colors.

Key Features:

  • Mobile-first responsive design
  • Indonesian Independence Day color scheme
  • Comprehensive component library
  • KTP scanning specific components
  • Form validation and biometric elements
  • Consistent animation and interaction patterns

Usage: Include perlinsos-styles.css in your HTML and use the classes as demonstrated above.
