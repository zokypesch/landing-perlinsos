Persetujuan Pengecekan Data Pribadi

Untuk keperluan pendaftaran Program Keluarga Harapan (PKH) melalui Portal Perlinsos, dengan ini saya:

MENYETUJUI

dilakukannya pengecekan dan verifikasi terhadap data pribadi saya yang akan diakses dari berbagai sumber data sesuai dengan ketentuan peraturan perundang-undangan yang berlaku, meliputi:

  • Data BPNT/TNI/Polri: Melalui sistem Badan Kepegawaian Negara (BKN).
  • Data Pendidikan: Melalui basis data Direktorat Jenderal Pendidikan Anak Usia Dini, Pendidikan Dasar, dan Pendidikan Menengah (Dikdasmen).
  • Data Kesehatan/Sertifikat Tenaga Kesehatan: Melalui basis data Kementerian Kesehatan (Kemenkes).
  • Data Kependudukan: Melalui basis data kependudukan Dukcapil.

Persetujuan ini diberikan untuk memastikan keakuratan data dan kelayakan saya sebagai calon penerima manfaat program PKH.
